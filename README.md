# Simple Interest Calculator

## Project Description
This project is a simple interest calculator implemented using a Bash script. 
It allows users to calculate simple interest based on the principal amount, rate of interest, and time period entered by the user.

## How It Works
The script prompts the user to enter:
- Principal amount
- Rate of interest
- Time period

It then calculates simple interest using the formula:

Simple Interest = (Principal × Rate × Time) / 100

## Technologies Used
- Bash scripting
- Git and GitHub for version control

## Usage
Run the script using:
bash simple-interest.sh

Follow the prompts to enter the required values.
